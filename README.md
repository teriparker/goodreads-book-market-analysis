# Goodreads Book Market Analysis
## Project Status: In Progress
This project is currently under development. The database schema, data cleaning process, and SQL analyses are being expanded. Additional dashboards and business analyses will be added as the project progresses.

## Project Overview
The objective of this project is to determine which fantasy subgenres demonstrate the strongest potential based on historical publishing trends, reader ratings, review activity, and overall market engagement. 

This project demonstrates an end-to-end data analytics worlfow including data cleaning, relational database designed, SQL analysis, exploratory data analysis, and interactive dashboard development in Tableau.

**Planned Analysis**
* Fiction vs. nonfiction trends
* Fantay subgenre analysis
* Publication trends
* Rating distributions
* Genre growth over time

## Key Business Questions
**Overall Market**
* How has book publishing changed over time?
* Which genres dominate the market?
* Which genres receive the highest reader engagement?

**Fiction Market**
* What percentage of books belong to the fiction market?
* Which fiction genres receive the highest average ratings?
* How has ficition publishing eveolved over time?

**Fantasy Market**
* What share of the fiction market is fantasy?
* How has fantasy publishing grown over time?
* Which fantasy authors ND BOOKS GENERte the strongest reader engagement?

**Fantasy Subgenres**
* Which fantasy subgenres are the most common?
* Which subgenres receive the highest average reader ratings?
* Which subgenres generate the greatest reader engagement?
* Which subgenres appear to represent potential growth opportuntites for publishers?
