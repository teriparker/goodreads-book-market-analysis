# Goodreads Book Market Analysis
## Project Status: In Progress

This project is currently under development. The database schema, data cleaning process, and SQL analyses are being expanded. Additional dashboards and business analyses will be added as the project progresses.

## Project Overview
Analyze Goodreads book data to understand publishing trends, genre popularity, and reader engagement. This project combines SQL data cleaning, exploratory analysis, and Tableau dashboards to answer business questions about the book market.

Planned Analysis
* Fiction vs. nonfiction trends
* Fantay subgenre analysis
* Publication trends
* Rating distributions
* Genre growth over time

## Dataset
Goodreads (2024)
https://www.kaggle.com/datasets/dk123891/books-dataset-goodreadsmay-2024

## Tools
* PostgreSQL
* Tableau
